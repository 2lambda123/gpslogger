Un registrador de datos simple y de ahorro de energía del sistema GPS. El propósito de esta aplicación es registrar sus coordenadas GPS a intervalos establecidos en un archivo en la tarjeta SD. Esta aplicación se ejecuta en segundo plano, por lo que puede ejecutarla durante una caminata larga, una caminata, un vuelo o una sesión de fotos, o incluso cuando solo va a la tienda a comprar leche, por ejemplo. Una vez que regrese a su computadora, puede usar los archivos resultantes para geoetiquetar fotos, enviar los datos a sitios de viajes o verlos usando Google Earth.


Para obtener más información, consulte <a href="https://gpslogger.app">https://gpslogger.app</a>

<b>Nota</b>: Recuerde configurar el permiso <b>Permitir siempre</b> para esta aplicación en su sistema operativo para que pueda registrar la ubicación en segundo plano.

<b>Características y funciones</b>

<ul>
<li>Puede grabar en formatos GPX, KML, CSV, NMEA y CSV. También puede grabar usando una URL de usuario o solo con visualización en la pantalla.</li>
<li>Registra puntos GPX como rutas y segmentos de ruta</li>
<li>También registra la velocidad, la dirección y la altitud, si están disponibles.</li>
<li>Permite agregar una descripción al punto, utilizando el menú de notas y anotaciones, se agregan como waypoints</li>
<li>Permite configurar intervalos de tiempo entre puntos</li>
<li>Es posible el filtrado de distancia mínima entre puntos o el filtrado centrado en la precisión</li>
<li>Icono de notificación con coordenadas y acciones rápidas</li>
<li>Uso de ocultadores de señales móviles para la ubicación cuando el GPS está desactivado</li>
<li>Selección de red selectiva, GPS y proveedores de localización pasivos</li>
<li>Envíe automáticamente el archivo al servidor o por correo electrónico, a intervalos establecidos. El envío es posible a la dirección de correo electrónico, servidor FTP, Dropbox, OpenStreetMap, Google Drive, URL de usuario.</li>
<li>Comparta la ubicación o registre archivos a través de SMS/correo electrónico (y otras aplicaciones como Facebook o Twitter, dependiendo de lo que haya instalado)</li>
<li>Empezar a grabar cuando se inicie el sistema operativo del teléfono</li>
<li>Invocar acciones usando el componente Tasker u otras aplicaciones de automatización.</li>
<li>Capacidad de crear un archivo con la configuración predeterminada para ser distribuido</li>
<li>Posibilidad de utilizar múltiples perfiles con diferentes configuraciones</li>
</ul>

Envíe errores, solicitudes de funciones y preguntas en github.com/mendhak/gpslogger (seguimiento de problemas).  

<b>Notas</b>

Esta aplicación solo usa una conexión de datos cuando configura el envío automático o funciones para una URL de usuario

Aunque la aplicación le permite establecer el intervalo en 0 segundos, no se recomienda porque la grabación y la grabación son demasiado rápidas. Tal configuración puede hacer que la aplicación se vuelva inestable o se bloquee. Recomendamos establecer un intervalo de 1 a 3 segundos.


<b>Explicación de autorizaciones</b>

Almacenamiento: lectura y escritura de archivos en la carpeta de la aplicación GPSLogger en la tarjeta SD

Comunicación de red: se utiliza al enviar archivos (Dropbox, Openstreetmap) o al enviar correos electrónicos o autorizar con Dropbox, Openstreetmap

Su ubicación: se utiliza para determinar su ubicación mediante GPS o buscadores de señales celulares

Herramientas del sistema (inicio automático al inicio): se usa cuando elige ejecutar una aplicación cuando se inicia el sistema operativo

Permisos de almacenamiento compartido: en Android 11, se usan para escribir archivos en rutas de archivos especificadas por el usuario
